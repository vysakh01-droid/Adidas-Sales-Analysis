# Adidas Sales Analysis

## Project Overview
This project focuses on analyzing Adidas sales data to uncover key business insights, trends, and performance metrics. The analysis combines data processing, querying, and visualization techniques.

## Tools & Technologies Used
- Jupyter Notebook (Python)
- SQL
- Power BI
- Pandas, NumPy, Matplotlib, Seaborn

## Project Objectives
- Clean and preprocess sales data
- Perform exploratory data analysis (EDA)
- Answer business questions using SQL queries
- Visualize insights using Power BI dashboards

## Files in This Repository
- MAIN PROJECT.ipynb – Python-based data analysis and visualization
- Adidas Sales Project.pdf – Project documentation/report
- Sales AI Format.pdf – Supporting project format/document
- README.md – Project overview and details

## Key Insights
- Sales trends over time
- Product and region-wise performance
- Revenue and profit analysis
- Data-driven recommendations for business growth

## Conclusion
This project demonstrates practical data analytics skills using Python, SQL, and Power BI, suitable for real-world business analysis and portfolio presentation.
